## Simple Memory Based BlockChain with POW

This project is created in Go.
Implementing core blockchain concept for learning purpose.

Concept learned -

1. Proof of Work
2. Handle Multiple connection
3. Validity of Block
4. Generation of Block
5. Consensus Algo

`nc localhost 8080`
